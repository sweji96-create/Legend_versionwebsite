# Logo Files Instructions

## Required Logo Files

### 1. Nashco Global Logo (`nashco-logo.png`)
- **Source**: https://github.com/user-attachments/assets/7da92b95-fc4f-49a4-975f-35360001e55d
- **Description**: Green and blue globe with "NASHCO GLOBAL" text in red
- **Format**: PNG with transparent background recommended
- **Dimensions**: Recommended minimum width: 200px
- **Usage**: Header navigation and footer branding

### 2. Zahy Group Logo (`zahy-group-logo.png`)
- **Source**: https://github.com/user-attachments/assets/760b9dd7-e561-4a62-a4b5-6bcbe998d6cf
- **Description**: Circular logo with horse/chess knight design and "ZAHHY GROUP" text
- **Format**: PNG with transparent background recommended  
- **Dimensions**: Recommended width: 180px
- **Usage**: Zahy Group section and footer branding

## How to Add Logos

1. Download the logo images from the URLs above
2. Save them as:
   - `nashco-logo.png`
   - `zahy-group-logo.png`
3. Place both files in the `/images` directory
4. The website will automatically display them in:
   - Header navigation bar
   - Zahy Group section
   - Footer
