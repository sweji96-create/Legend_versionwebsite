# Nashco Global Corporate Website

Professional single-page website showcasing Nashco Global's excellence in chemicals (industrial refrigerants via Zahy Al Khaleej LLC), automotive spare parts, manufacturing, and trading across the GCC.

## Features
- Responsive design
- Image gallery using existing assets
- SEO optimized for GCC industry keywords
- Contact form
- Navigation with anchor links

## Technologies
HTML5, CSS3, Vanilla JavaScript

## Assets
All visuals use only pre-existing images in /assets/ folder.

## Local Preview
