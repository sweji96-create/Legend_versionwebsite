# Legend Version Website

## Overview
Welcome to the `Legend Version Website` repository! This repository contains the assets and source code for a modern multi-page website built to highlight professional features while maintaining a responsive and accessible design. The website has undergone significant updates from **PR #16**, adopting modular architecture with a mobile-first approach.

## Website Features

### Pages

1. **Home Page (index.html)**
   - Hero Section: Features a rotating background image highlighting themes of progress and innovation.
   - Division Cards: Display cards for all website divisions with fadeIn animations.
   - Video Section: Displays looping GCC flags video.

2. **About Page (about.html)**
   - Dedicated to corporate background, history, and achievements.
   - Background: Used Dubai skyscraper imagery with parallax scrolling.

3. **Zahy Al Khaleej (zahyalkhaleej.html)**
   - Displays Zahy Group logo and affiliated division highlights.

4. **Trading Page (trading.html)**
   - Displays operations by region. Includes text animations.
